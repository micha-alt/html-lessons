Beispiel eines HTML-Dokumentes

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>HTML Schulung</title>
  </head>
  <body>
    <h2>HTML Schulung</h2>
    <p>Allgemeine Regeln der HTML-Programmierung</p>
  </body>
</html>

Elemente/Tags

HTML-Dokumente bestehen aus Text, der in Tags zur Auszeichnung eingeschlossen ist. Der Tag beginnt mit einem "Kleiner-Als"-Zeichen < und endet mit dem "Größer-Als" > Der Tagname darf keine Leerzeichen enthalten. Das Schliessen des Tags geschieht durch einen Endtag </tagname> 

<p>Beispieltext</p>

Leere Elemente/Tags

HTML definiert auch Tags, die keinen eigentlichten Content/Inhalt haben. Beispiel: img Ein solcher Tag kann auch "offen" gelassen werden. Oder als leeres Element ausgezeichnet werden. 

<img>
<img></img>
<img />

Attribute

Ein Tag kann zusätzliche Informationen in sogenannten Attributen bereitstellen. Der Attribut-Wert ist (nicht zwingend) in Hochkommata oder Anführungszeichen einzuschließen 

<img src="sample.jpg">
