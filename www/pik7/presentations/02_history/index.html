HTML5

Die WHATWG (gegründet von Mozilla, Apple, Opera - später Google) formierte sich 2004 als Gegenentwurf zur W3C, um die Entwicklung von HTML weiter zu treiben. Sie formulierte eine eigene Spezifikation HTML (vorher HTML5) für die nächste Version als lebendiges Dokument - Living Standard.

W3C beginnt ab 2008 die Entwicklung von HTML5, veröffentlicht einen ersten Working Draft, und geht von einer Spezifikation im Jahre 2014 aus. Dabei übernimmt sie die Hauptbestandteile der WHATWG.
